# RECKON

**RECKON** is a modern scientific calculator built with HTML, CSS, and Vanilla JavaScript. It supports advanced mathematical operations, variable assignments, trigonometric functions, and a ledger-style calculation history, all wrapped in a clean and responsive user interface.

## Features

* Scientific calculations
* Variable assignment and reuse (`x = 12`, `x * 2`)
* Trigonometric functions (`sin`, `cos`, `tan`)
* Advanced math functions (`sqrt`, `log`, `factorial`, `gcd`, `hypot`, etc.)
* Mathematical constants (`pi`, `e`, `tau`)
* Ledger-style history panel
* Click previous results to reuse them
* Keyboard shortcuts and history navigation
* Responsive design for desktop and mobile

## Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
